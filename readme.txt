== Description ==
WP Related Posts help you display simply related posts in a single post. It's easy to use and lightweight.

To display simply related posts under a single post, use this shortcode in single.php file: 
`
<?php echo do_shortcode( '[wp_related_posts id="1"]' ) ?>
`
